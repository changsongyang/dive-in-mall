2. ES6箭头函数保持this的指代是不变的

```javascript
        wx.request({
            url: `${config.apiBaseUrl}theme/by/names`,
            method: 'GET',
            data: {
                names: 't-1'
            },
            header: {
                appkey: `${config.appKey}`
            },
            // 箭头函数保持this的指代不变
            success: res => {
                this.setData({
                    topTheme: res.data[0]
                })
            }
        })
```
